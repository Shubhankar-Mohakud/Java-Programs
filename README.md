# JAVA Lab

This folder contains a collection of Java lab exercises and practice programs organized by lab date.

## Folder Layout

- `Main.java` - top-level sample program.
- `Lab 1 11_Dec_24/` through `Lab10_19Mar25/` - lab-wise Java questions and examples.
- `Lab Test/` - test programs and thread examples.

## How to Run

You can compile and run any Java file from PowerShell with:

```powershell
javac FileName.java
java FileName
```

If a file is inside a folder, first change into that folder:

```powershell
cd "Lab 9 19_Feb_25"
javac Q1.java
java Q1
```

## Notes

- Some folders contain multiple question files for a single lab.
- A few files may depend on console input, file handling, threads, or Swing.
- Compiled `.class` files and IDE files are present in the workspace but are not required to run the source code.
